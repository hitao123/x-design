# 介绍

X Design 是一个基于 Vue 3 的企业级组件库，专为 PC 端应用设计。

## ✨ 特性

- 🎨 **精美设计**：遵循现代设计规范，提供美观且一致的用户体验
- 🔧 **开箱即用**：高质量的 Vue 3 组件，满足企业级应用需求
- 🎯 **TypeScript**：完整的 TypeScript 类型定义
- 🎨 **主题定制**：基于 SCSS 的主题系统，轻松定制品牌风格
- 📦 **按需引入**：支持 Tree Shaking，减少包体积
- 🌍 **国际化**：内置多语言支持

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **语言**：TypeScript
- **构建**：Vite
- **样式**：SCSS
- **测试**：Vitest
- **文档**：VitePress

## 📦 包管理

X Design 采用 Monorepo 架构，包含以下子包：

- `@x-design/components` - 组件库主包
- `@x-design/icons` - 图标库（独立包，参考 Ant Design Icons）
- `@x-design/theme` - SCSS 主题系统
- `@x-design/utils` - 工具函数库
